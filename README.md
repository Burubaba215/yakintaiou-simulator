# 申し送りシミュレーター（介護）

介護現場の新人研修向けWebアプリです。  
5名の仮想入居者の申し送りを確認し、未排便・水分強化・転倒注意などの対応を選択します。

## 機能

- 5名分の申し送り表示
- ブラウザ音声読み上げ
- 対応選択クイズ
- 採点と解説
- スマホ対応

## 想定対応

- 未排便確認・ピコスルファート服用対応
- 水分強化
- 転倒注意・見守り強化
- 食事観察・義歯確認
- 家族面会の申し送り

## 公開方法

1. GitHubで新規リポジトリを作成
2. このフォルダの `index.html` `style.css` `script.js` `README.md` をアップロード
3. Settings → Pages → Branch を `main` / `/root` に設定
4. 数分後にGitHub Pagesで公開

## 注意

このアプリは学習・研修用です。実際の服薬対応は、所属事業所の手順、医師・看護師の指示、記録確認に従ってください。
