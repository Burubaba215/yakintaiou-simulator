# 介護申し送りトレーニング（仮）

介護現場の新人教育を想定したWebアプリ研修用のミニ教材です。  
夜勤入りの申し送りを聞き、入居者さんの状態から必要な対応を選びます。

登場する名前・内容はすべて仮設定です。仕事内容の雰囲気や、申し送りで見るポイントが伝わることを目的にしています。

## 機能

- 5名分の申し送り表示
- ブラウザ音声読み上げ
- 対応選択クイズ
- 採点と解説
- スマホ対応

## 想定対応

- 未排便確認・ピコスルファート服用対応
- 水分強化
- 転倒注意・見守り強化
- 食事観察・義歯確認
- 家族面会の申し送り

## 公開方法

1. GitHubで新規リポジトリを作成
2. このフォルダの `index.html` `style.css` `script.js` `README.md` をアップロード
3. Settings → Pages → Branch を `main` / `/root` に設定
4. 数分後にGitHub Pagesで公開

## 注意

このアプリは学習・研修用です。実際の服薬対応は、所属事業所の手順、医師・看護師の指示、記録確認に従ってください。
